## 🚀 快速开始

### 1. 克隆仓库
```bash
git clone https://github.com/SamuelPang/TRnews.git
cd TRnews
```

### 2. 查看今日内容
```bash
# 查看最新报告
ls -lt data/
ls -lt reports/
ls -lt platforms/
```

### 3. 发布内容
```bash
# 运行发布脚本
bash scripts/daily-publish.sh
```

## 📝 使用指南

### 检索历史内容
```bash
# 按日期查找
find data -name "YYYY-MM-DD" -type d

# 按平台查找
find platforms -name "*.md"

# 按关键词搜索

```

### 添加新内容
```bash

mkdir -p reports/YYYY-MM-DD
mkdir -p platforms/xiaohongshu/YYYY-MM-DD

# 复制文件

cp news-YYYY-MM-DD.json data/YYYY-MM-DD/
```
